# Due Diligence Checklist

## Corporate
- [ ] Certificate of incorporation / formation documents
- [ ] Bylaws / operating agreement (current)
- [ ] Cap table and equity summary
- [ ] Board minutes (last 2-3 years)
- [ ] Written consents
- [ ] Good standing certificates
- [ ] Foreign qualifications
- [ ] Organizational chart (legal entities)

## Material Contracts
- [ ] Customer agreements (top 10 by revenue)
- [ ] Vendor/supplier agreements (top 10 by spend)
- [ ] Partnership / channel agreements
- [ ] Lease agreements
- [ ] Loan / credit agreements
- [ ] License agreements (inbound and outbound)
- [ ] Government contracts

## Intellectual Property
- [ ] Patent portfolio (granted and pending)
- [ ] Trademark registrations
- [ ] Copyright registrations
- [ ] Domain names
- [ ] Open source usage and compliance
- [ ] IP assignment agreements (employees and contractors)
- [ ] Inbound license agreements
- [ ] Outbound license agreements

## Litigation & Disputes
- [ ] Pending litigation
- [ ] Threatened claims
- [ ] Settled matters (last 3-5 years)
- [ ] Arbitration proceedings
- [ ] Regulatory investigations or inquiries
- [ ] Outstanding judgments or orders

## Regulatory & Compliance
- [ ] Licenses and permits
- [ ] Regulatory filings
- [ ] Compliance program documentation
- [ ] Past regulatory examinations / audits
- [ ] Consent orders or enforcement actions
- [ ] Sanctions screening results

## Employment
- [ ] Employee headcount and org chart
- [ ] Key employee agreements
- [ ] Equity compensation plans and grants
- [ ] Contractor agreements
- [ ] Employee handbook / policies
- [ ] Pending or threatened employment claims
- [ ] Non-compete / non-solicitation agreements
- [ ] Contractor classification assessment

## Data Privacy & Security
- [ ] Privacy policy
- [ ] Data processing agreements
- [ ] Data breach history
- [ ] Security certifications (SOC 2, ISO 27001)
- [ ] Data flow mapping
- [ ] GDPR/CCPA compliance assessment

## Insurance
- [ ] Current policies (D&O, E&O, cyber, GL, EPLI)
- [ ] Claims history
- [ ] Coverage adequacy assessment

## Financial
- [ ] Audited financial statements (last 3 years)
- [ ] Outstanding debt and liens
- [ ] Material guarantees
- [ ] Tax returns and any pending audits

## Tax
- [ ] Tax returns (federal, state, international — last 3 years)
- [ ] Pending tax audits or disputes
- [ ] Transfer pricing documentation (if applicable)
- [ ] Tax opinions or rulings

# NDA Screening Checklist

Quick-reference checklist for NDA triage. For full methodology, see `playbooks/nda-triage.md`.

## Screening Items

- [ ] **Structure**: Mutual or unilateral? Matches intent? Standalone?
- [ ] **Affiliate coverage**: Covers your organization's affiliates? Mutual treatment?
- [ ] **Definition scope**: Reasonable definition with proper exclusions (4 for mutual, 3 for one-way)?
- [ ] **Obligations**: Use restriction? Reasonable care standard? Reverse engineering prohibition?
- [ ] **Carveouts**: All required carveouts present?
- [ ] **Compelled disclosure**: Permitted with notice? Whistleblower safe harbor?
- [ ] **Term**: Reasonable duration? Survival period (3-5 years mutual, indefinite one-way)?
- [ ] **Return/destruction**: Obligation to return or destroy on termination?
- [ ] **Remedies**: Injunctive relief without bond? Attorneys' fees?
- [ ] **Problematic provisions**: Non-compete? Non-solicitation? Exclusivity? IP assignment? Residuals?
- [ ] **Governing law**: Reasonable jurisdiction? Assignment restrictions acceptable?

## Classification

- **GREEN**: All items pass → route for signature
- **YELLOW**: 1-3 minor deviations, no RED → single review pass
- **RED**: Any RED item → full review, negotiation, or counterproposal

# Board & Governance Playbook

## When This Applies

When preparing board materials, drafting resolutions or consents, taking minutes, or handling corporate governance matters.

## Required Context

- **Always**: `core/principles.md`, `core/organization.md`
- **Legal knowledge**: `reference/legal-knowledge/corporate-and-governance.md`

## Common Tasks

### Board Resolutions
- Identify what requires board approval (major contracts, equity grants, M&A, officer appointments, significant expenditures)
- Draft resolution with clear recitals and operative provisions
- Circulate for review and approval
- Written consent (in lieu of meeting) vs. meeting vote

### Board Minutes
- Record date, time, attendees, quorum confirmation
- Summarize discussion (not verbatim transcript)
- Record motions, votes, and resolutions adopted
- Note any recusals or abstentions
- Attach or reference any materials presented

### Officer Appointments / Changes
- Board resolution appointing or removing officers
- Update corporate records
- File any required state/federal updates
- Update signing authority documentation

### Entity Formation
- Choose jurisdiction and entity type
- File formation documents
- Obtain EIN
- Open bank accounts
- Adopt organizational documents (bylaws, operating agreement)
- Qualify in foreign jurisdictions as needed

### Annual Compliance
- Annual reports and franchise tax filings
- Officer/director updates
- Registered agent maintenance
- Foreign qualification renewals
- Good standing certificates as needed

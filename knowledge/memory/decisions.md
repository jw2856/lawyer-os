# Decision Log

> Record significant legal decisions with rationale. This builds institutional memory
> so future decisions can reference past reasoning.

## Format

```
### [Date] — [Decision Title]
**Context**: What prompted this decision
**Decision**: What was decided
**Rationale**: Why
**Alternatives considered**: What else was on the table
**Outcome**: How it played out (add later)
```
